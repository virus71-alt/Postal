# Monthly SEO Progress Report

This file defines monthly reporting for ongoing SEO campaigns.

---

## 1. Purpose

Monthly reporting should show:

```text
What changed?
What improved?
What was completed?
What still blocks growth?
What happens next?
```

---

## 2. Required Sections

```text
1. Month Summary
2. Work Completed
3. Keyword Movement
4. GSC Performance
5. GA4 / Conversion Performance
6. GBP Performance if local
7. Content Published
8. Technical Fixes
9. Backlinks / Authority Work
10. AEO/GEO Progress
11. Issues / Risks
12. Next Month Plan
```

---

## 3. KPI Table

```text
Metric
Previous Month
Current Month
Change
Notes
```

Use for:

```text
GSC clicks
GSC impressions
Average CTR
Average position
Organic sessions
Conversions
Phone/WhatsApp clicks
GBP calls/directions
Top 10 keywords
Referring domains
Indexed pages
```

---

## 4. Reporting Rule

Do not hide bad news.

If rankings did not move, explain why:

```text
Implementation delay
Indexing delay
Authority gap
Competitive SERP
Content not published
Tracking issue
Algorithm volatility
```

Then give next action.
