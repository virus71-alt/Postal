## Lightweight Specialist Playbooks

Some files are intentionally shorter. They act as routing/playbook files so the skill is universal and complete across industries and workflows. Expand them later only when a specific client/project needs more detail.

Examples:

- industry-playbooks/*
- reporting/* specialist reports
- selected strategy files
- selected competitor/local/backlink submodules

This is intentional for a v1 universal skill: deep where agents need operating logic, concise where agents need routing and checklist behavior.
