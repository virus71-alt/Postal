# Map Pack Audit

This file defines how Beyond SEO evaluates local map-pack competition and opportunity.

Map-pack visibility is often the fastest path to calls for local businesses.

---

## 1. When to Use

Use for local intent keywords such as:

```text
service near me
service + city
best service + city
emergency service + city
provider type + city
```

---

## 2. Data Sources

Use:

```text
Google search/manual map checks
Apify Google Maps actors
GBP screenshots/export
Local Falcon/BrightLocal exports if provided
SERP screenshots
Competitor GBP profiles
```

---

## 3. What to Record

For each map keyword:

```text
Query
City/location
Date
Client position if visible
Top 3 businesses
Top 10 businesses
Primary categories
Ratings
Review counts
Website links
Distance/location pattern
Services shown
Ads/local service ads if present
```

---

## 4. Map Competitor Analysis

Compare:

```text
category match
review count
rating
review keywords
profile completeness
photos
posts
services
website authority
local landing page quality
citations/local links
```

---

## 5. Map Pack Gap Types

```text
Category gap
Review gap
Distance/location disadvantage
Service listing gap
Photo/profile completeness gap
Website landing page gap
Citation/NAP gap
Authority gap
```

---

## 6. Map Pack Action Plan

Actions may include:

```text
fix GBP categories
expand services
build review workflow
add service photos
publish GBP posts
answer Q&A
create matching local service pages
clean citations
build local links
add UTM tracking
```
